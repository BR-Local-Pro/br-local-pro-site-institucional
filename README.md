# BR Local Pro - Site Institucional

Site institucional da BR Local Pro, criado para converter visitantes em conversas no WhatsApp.

## Estrutura

- `index.html` - pagina principal
- `styles.css` - estilos do site
- `site.js` - interacoes e carrossel
- `assets/` - imagens do site
- `vendor/` - bibliotecas locais usadas pela pagina

## Publicacao

Site estatico pronto para GitHub Pages.
